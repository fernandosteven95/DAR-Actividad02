Justificación

¿Para qué sirve?

Angular es ideal para desarrollar aplicaciones web dinámicas y escalables.

¿Qué aplicaciones tiene?

Desde aplicaciones web hasta móviles y progresivas (PWA).

¿Sobre qué lenguajes se apoya?

Principalmente en TypeScript, JavaScript, HTML y CSS.

¿Qué prerrequisitos necesita?

Conocimientos básicos de HTML, CSS y JavaScript.
