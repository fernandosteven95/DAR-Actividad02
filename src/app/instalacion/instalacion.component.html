Instalación de Angular

1. Requisitos previos

  • Tener Node.js instalado (versión LTS recomendada).
  • Conexión a internet para instalar dependencias.
  • Un editor de texto, como VS Code.

2. Instalación de Node.js

Descarga Node.js desde su sitio oficial e instala la versión LTS.

node -v

Resultado esperado: v16.x.x o superior.

3. Instalación de Angular CLI

Ejecuta el siguiente comando para instalar Angular CLI globalmente:

npm install -g @angular/cli

Verifica la instalación con:

ng version

Resultado esperado:


    Angular CLI: 15.x.x
    Node: 16.x.x

4. Versiones necesarias

  • Node.js: 16.x.x o superior.
  • Angular CLI: 15.x.x o superior.
  • npm: 8.x.x o superior.

5. Rutas para tener en cuenta

Por defecto, Angular crea las carpetas y archivos en el directorio actual desde donde se ejecuta el comando.

Ejemplo:

ng new my-angular-app

Esto creará un directorio llamado my-angular-app en la ruta actual.

6. Variables de entorno

Angular no requiere variables de entorno específicas para funcionar, pero puedes configurar un archivo environment.ts para almacenar configuraciones como URLs o claves API.

src/environments/environment.ts

7. ¿Necesita base de datos?

Angular no necesita una base de datos para funcionar, pero puedes integrarlo con APIs que interactúen con bases de datos como MySQL, MongoDB, o Firebase.
