Primeros Pasos con Angular

1. Crear un proyecto "Hola Mundo"

Sigue estos pasos para crear tu primer proyecto en Angular:

  1. Abre una terminal y ejecuta el siguiente comando para crear un nuevo proyecto: 
    ng new hola-mundo
  2. Angular te pedirá seleccionar algunas configuraciones. Por ejemplo:
    • ¿Agregar routing? Responde No.
    • ¿Qué formato de hojas de estilo deseas usar? Selecciona CSS.
  3. Navega al directorio del proyecto: 
    cd hola-mundo
  4. Inicia el servidor de desarrollo: 
    ng serve --open
    Esto abrirá el navegador en http://localhost:4200 mostrando la página de inicio del proyecto.

2. Estructura de un proyecto Angular

Un proyecto Angular tiene la siguiente estructura:

Carpetas principales

  • src/: Contiene el código fuente de la aplicación.
  • app/: Contiene los componentes y la lógica de la aplicación.
  • assets/: Recursos estáticos como imágenes y fuentes.
  • environments/: Configuración para diferentes entornos (desarrollo, producción).

Archivos principales

  • index.html: Punto de entrada principal de la aplicación.
  • styles.css: Estilos globales de la aplicación.
  • angular.json: Configuración del proyecto Angular.
  • package.json: Contiene las dependencias del proyecto.
  • tsconfig.json: Configuración del compilador TypeScript.
  • app.component.ts: Define la lógica y comportamiento del componente principal de la aplicación.
  • app.component.html: Contiene el código HTML para la interfaz del componente principal.
  • app.component.css: Define los estilos específicos (CSS) para el componente principal.
  • app.module.ts: Configura y registra los módulos y componentes utilizados en la aplicación.

3. Modificar el componente principal

El archivo app.component.ts es el componente principal de Angular. Modifícalo para mostrar un "Hola Mundo":


      /* app.component.ts */
      import { Component } from '@angular/core';

      @Component({
            selector: 'app-root',
            template: '<h1>Hola Mundo desde Angular</h1>',
        styleUrls: ['./app.component.css']
      })
      export class AppComponent { }

Guarda los cambios y verifica el resultado en el navegador.
